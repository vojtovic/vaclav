Základy JS - opakování

1: Co je JavaScript a k jakým účelům se používá?

Doplň vhodné odpovědi a využij k tomu nečíslovaného seznamu.

Použij v textu aspoň 3 internetové odkazy na zajímavé příklady využití JS!

JavaScript je skriptovací jazyk,interpretovany nejčastěji webovými prohlížeči.

JS není roven jazyku Java, oficiálně se nazývá ECMAScript. Existují různé verze, např. ES1 až ES6.

využití

  • tvorba interaktivních webových staránek
  • tvorba dinamickych stranek
  • tvoření aplikací na strane klienta i serveru

2: Jaká pravidla platí pro psaní kódu v JavaScriptu?

Napiš aspoň pět zásad ve podobě číslovaného seznamu.

Použij angličtinu!

  1. Ending statements with semicolon is not required, but highly recommended. ( ; )
  2. One place you will find statements grouped together in blocks, is in JavaScript functions. ( {...} )
  3. Code after double slashes // or between /* and */ is treated as a comment.
  4. All JavaScript identifiers are case sensitive. The variables "lastName" and "lastname", are two different variables.
  5. A JavaScript program is a list of programming statement, the statements are executed, one by one, in the same order as they are written.

Doplňte další možnosti ve formě kódu.

Opatřete kód poznámkami!

// 1. Vložit kód do tagu script

<a href="javascript:alert('To moc nedělej lol')">kouzelný "ne"odkaz</a>

4: Jak je možné v JavaScriptu deklarovat proměnné?

Uveďte na příkladech různé datové typy, které je možné využívat v JS.

Doplňte chybějící kód/text podle vzoru úvodních dvou řádků!

let x = 0; // s přiřazením hodnoty
let x; // bez přiřazení hodnoty
let x = null; // proměná x se vytvoří ale nemá žádnou hodnotu
let x = 69 // celé číslo (typ number)
let x = 4.20 // reálné číslo (typ number)
let x = 0.3e5 // zápis desetinného čísla pomocí exponentu
let a = 'a' // typ string
let prestupny = false // konstanta prestupny ma hodnotu false

5: Jaké operátory můžeme používat v JavaScriptu?

Vypište operátory, které patří do uvedených skupin.

U méně obvyklých operátorů dopište do závorky jejich význam!

Matematické

+, -, *, /, % (modulo - zbytek po celočíselném dělení), ** (umocnění)

Přiřazovací

+=, *=, %=, -=, /=, <<=, >>>=, >>=, &=, |=, ^=

Logické

&&, ||, !, (Logické operátory očekávají, že operandy jsou logického typu a vrací logickou hodnotu true nebo false)

Porovnávací

Bitové

&, |, ^, <<, >>, >>>

Typové

typeof (typ proměnné), instanceof (...)

6: Která popup okna lze využít v JavaScriptu?

K uvedeným příkladům připište do poznámky, co se stane na webové stránce.

Na konci doplňte aspoň dvě varování, proč bychom se měli vyhýbat použití popup oken v reálných webových aplikacích.

alert()

/*Vyskočí popup okno s textem "Nezadal jsi správné heslo!"*/
alert('Nezadal jsi správné heslo!');

prompt()

/* Do proměnné password vloží heslo zadané uživatelem */
var password = prompt('Zadej heslo zde');

confirm()

/*Vyskočí popup okno s textem "Chceš přejít na stránky Seznam.cz?" a s tlačítky OK / Cancel*/
if (confirm('Chceš přejít na stránky Seznam.cz?')) location.href='https://seznam.cz';
